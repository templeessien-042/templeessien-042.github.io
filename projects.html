Notes

What the store taught me.

Holding state without a backend forces a decision: where does the truth live? The answer here is a single cart array. Nothing reads the DOM to find out what's in the basket — the DOM is a rendering of the array, and it is thrown away and rebuilt whenever the array changes. It's the smallest possible version of the idea every front-end framework is built around.

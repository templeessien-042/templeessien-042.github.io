Computer Science undergraduate, matric AK22/PHS/CSC/042. I write front-end code the long way — by hand, in a plain text editor, until I understand what every line is doing.

How I work

Frameworks hide the machinery. For coursework, that's a bad trade — the point is to see the machinery. So both of these builds are raw HTML, CSS, and JavaScript, written in Notepad, styled from an empty stylesheet, and deployed as static files.

It's slower. It also means nothing here is a mystery to me.

What I'm learning

NetCentric computing sits between the browser and the network: how a document gets requested, routed, rendered, and how state survives the trip. The storefront in the companion build is my first pass at holding state in the client without a backend to lean on.

Toolkit

STYLE

BEHAVIOUR

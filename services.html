Capabilities

What I can build for you.

Static sites, storefronts, and the connective tissue between them. Small scope, done properly.

01

Static site development

Multi-page sites built from scratch. Hand-written markup, a single shared stylesheet, and navigation that stays consistent across every view. Deployed to GitHub Pages or Netlify at no hosting cost.

02

Client-side storefronts

Product grids, cart logic, live totals — running entirely in the browser. Suitable for small retailers who need a catalogue and a basket before they need a payment processor.

03

Responsive layout

Grid and Flexbox layouts that reflow cleanly from desktop to phone. Keyboard focus stays visible; reduced-motion preferences are respected.

04

Form integration

Contact and comment forms wired to a delivery service, so a submission actually reaches an inbox instead of vanishing into a dead action attribute.

Process

Three passes, in order.

PASS 01

Structure

Markup first, unstyled. If the page doesn't read correctly without CSS, the structure is wrong.

PASS 02

Style

Tokens defined once at the root, then applied. Colour and type decisions get made before a single component does.

PASS 03

Behaviour

JavaScript added last, and only where the page genuinely needs to change in response to a user.
